# Discord RAG FAQ Bot

A Discord chatbot that answers questions about the AI Bootcamp & Internship program using Retrieval-Augmented Generation (RAG) over official PDF documents.

**Tech:** Discord.py · LangChain · Qdrant Cloud · Ollama · Python

**How it works:**  
PDFs → chunking → embeddings → vector search → LLM response

**Behavior:**  
Answers only from documents. Says *“I don’t know”* if information is not found.
